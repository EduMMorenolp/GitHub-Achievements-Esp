# 🏅  Mostrando logros 🏅

#### Mostrar los logros en tu perfil es completamente opcional; de forma predeterminada, pueden ser vistos por cualquier persona que vea tu perfil público.

#### Puedes optar por no mostrar los logros en tu perfil yendo a la configuración de tu perfil. [profile settings](https://github.com/settings).
